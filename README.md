# can-landing

Clean, modern, bilingual single-page personal landing page for Hamdi Can Ernalbantoğulları's QA Engineer, Test Automation Engineer, and SDET job search.

## Overview

This project is a deployment-ready static site built with plain HTML, CSS, and minimal JavaScript. It includes English and Turkish language options, a professional hero section, profile summary, core QA skills, experience highlights, projects, contact labels, and basic SEO/Open Graph metadata.

## Files

- `index.html` - page content and SEO metadata
- `style.css` - responsive dark-neutral visual design
- `script.js` - language switching, current year, smooth scrolling, and email copy helper
- `README.md` - project notes

## Deploy

Host the folder with any static hosting provider, such as GitHub Pages, Netlify, Vercel, Cloudflare Pages, or a simple static web server.

Keep landing page copy aligned with the approved source content before publishing.
